# SUMRRRSALT

**One Image → Every Platform**

Stop manually resizing images for social media. Upload once, export pixel-perfect assets for 15+ platforms instantly.

## Features

- **Multi-Step Scaling** — Pro-grade 50% step downscaling prevents aliasing artifacts. Your 16px favicons look crisp, not blurry.
- **Auto Sharpening** — Convolution-based sharpening on small assets keeps every icon sharp at any size.
- **Smart Crop** — Automatic center-crop with aspect ratio preservation. No more stretched or squished images.
- **Privacy First** — Everything runs in your browser. Your images never leave your device.

## Getting Started

### Prerequisites

- Node.js 18+ or Bun

### Installation

```bash
# Clone the repository
git clone https://github.com/keeganmoody33/sumrrrsalt.git
cd sumrrrsalt

# Install dependencies
npm install
# or
bun install

# Run the development server
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to use the app.

## How It Works

1. **Upload** your image (drag & drop or click to browse)
2. **Select** the platforms you need assets for
3. **Download** all perfectly-sized assets as a ZIP
